How to Use Send and Archive

This extension provides three ways to send a message and archive the original:

  • Toolbar Button: Click the "Send and Archive" button in the compose window toolbar
  • Keyboard Shortcut: Press Ctrl+Alt+Enter in any compose window
  • Context Menu: Right-click the toolbar button for additional options

Note: The original message will only be archived when you're replying to or forwarding an existing message. New messages have no original to archive.

Archive Settings: This extension uses Thunderbird's built-in archive functionality, which respects your existing archive preferences (Tools → Account Settings → Copies & Folders → Message Archives).
